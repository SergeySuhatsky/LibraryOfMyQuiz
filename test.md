### **ПРОГРАММА 1: Сравнение методов на задачах аппроксимации 2D функций**

**Задачи:**
1. **Задача A** — Аппроксимация целевой функции рельефа:
   $$f(x, y) = \frac{e^{-(x^2+y^2)}}{1 - x - y}$$ 
   на области с барьером $x + y = 1$

2. **Задача B** — Аппроксимация потенциальной энергии молекулы:
   $$V(x, y, z) = \frac{e^{-(x-z_0)^2-(y-w)^2-(z-z_0)^2}}{x \cdot y} + D\left(1-e^{-\alpha(x-z_0)}\right)^2 + D\left(1-e^{-\alpha(y-w)}\right)^2$$

**Метрики для каждого метода:**
- Относительная ошибка на сетке точек
- Время выполнения (в мс)
- Количество узлов полюсов
- Стабильность вблизи барьера

**Выводы в Программе 1:**
1. 2 графика поверхности (точная функция vs аппроксимация)
2. Карты ошибок с контурами
3. Таблица сравнения метрик (метод | ошибка | время | узлы)
4. График зависимости ошибки от времени выполнения
5. Анализ сходимости вблизи особенностей

### **ПРОГРАММА 2: Сравнение методов на задачах оптимизации с ограничениями**

**Задачи:**
1. **Задача A** — Минимизация интегрального функционала (движение поисковых частиц):
   $$J[\gamma] = \int_0^1 f(t(x), t(y)) dt$$
   с барьером $x + y < 1$

2. **Задача B** — Поиск оптимальной траектории через двумерный ландшафт (A → B)
   с ограничением: траектория не пересекает линию $x + y = 1$

**Метрики для каждого метода:**
- Значение минимума функционала
- Время поиска оптимума (в мс)
- Количество итераций
- Эффективность обхода барьера

**Выводы в Программе 2:**
1. Контурный график функции с найденной траекторией (все 6 методов)
2. Поверхность функционала с минимумом
3. Таблица сравнения: метод | минимум | время | итерации
4. График сходимости для каждого метода
5. Визуализация эффективности обхода барьера

### **КАЖДЫЙ МЕТОД ВКЛЮЧАЕТ:**

1. **p-AAA** — построение рациональной функции через тензорное разложение
2. **Low-rank-AAA** — низкоранговое разложение с итеративной оптимизацией
3. **Сантана-Кёрнера** — стабилизированная итерация с ортонормированными базисами
4. **Рациональные схемы с кластеризацией** — группировка полюсов для устойчивости
5. **Классический Паде** — разложение по полиномам Паде
6. **Вложенный Паде** — иерархическое разложение Паде

### **ИТОГОВЫЙ АНАЛИЗ:**

Каждая программа выведет:
- ✓ Визуализацию точных функций
- ✓ Визуализацию полученных аппроксимаций
- ✓ Карты абсолютных и относительных ошибок
- ✓ Таблицу метрик (ошибка, время, устойчивость)
- ✓ Кривые Парето (ошибка vs время)
- ✓ Выводы о лучших методах для каждого класса задач

Таким образом, сравнительный анализ покажет, какие методы наиболее эффективны для аппроксимации с особенностями, какие дают минимальное время вычислений, и какие наиболее устойчивы вблизи барьеров.
